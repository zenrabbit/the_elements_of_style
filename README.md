# Elements of Style Rule Lookup
A simple interactive tool for rule look-ups from this classic 'little book'.
This is a lightweight browser-based look-up tool for the 18 core rules from The Elements of Style (public domain Strunk version).

## Features
- Search rules instantly
- Examples and explanations
- Works offline
